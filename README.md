# recursion-in-c

 C Recursion Programs

## Overview

This repository contains C programs demonstrating the concept of recursion. Recursion is a programming technique where a function calls itself to solve smaller instances of a problem until a base condition is reached.

These programs help in understanding recursive thinking, function calls, stack behavior, and problem-solving techniques.

The programs are useful for:

- Beginners learning recursion in C
- College laboratory exercises
- Coding interview preparation
- Logic-building practice
- Understanding recursive algorithms

### 3. Sum of First N Natural Numbers

This program calculates the sum of numbers from 1 to N recursively.

#### Output Example

```text

```

For:

```text
N = 5
```

Calculation:

```text
5 + 4 + 3 + 2 + 1 = 15
```

#### Concepts Used

- Recursive return values
- Mathematical recursion
- Problem decomposition
